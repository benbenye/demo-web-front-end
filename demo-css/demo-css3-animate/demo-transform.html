		css3的transition和transform必要的两个属性
		transition对标签的变化过程进行设置。
		transition-property 要变化的属性
		transition-duration 变化的速度 单位s
		transition-timing-function 贝塞尔曲线的方式渐变
		transition-delay 是否延迟执行变化 单位s
		transform可以设置的函数
		transform-origin:百分值,em,px 元素基准点
		rotate 对元素进行旋转 单位deg
		translate 对元素平移 单位px x和y
		scale 对元素大小缩放 x和y
		skew 斜切变化 扭曲 单位deg 角度
		matrix 矩阵变形待续。。。
